Predicate Bootcamp

The journey at Predicate starts with Bootcamp - a foundational experience in Computer Science. We cover the basics of working with the Raspberry Pi, using the Linux operating system, writing code, digital circuits, and problem decomposition. These skills are the necessary building blocks for the rest of our courses at Predicate.

We love Linux. Predicate Academy teaches the Linux operating system, as opposed to Mac or Windows, due to its accessibility and wide adoption in the computing field. Rest assured, we have designed the Bootcamp program with the expectation that students have no prior experience working with Linux.

Core concepts covered in the 10-week Bootcamp program include:

  • Computing Hardware - understanding the various parts of the Raspberry Pi
  • Using Linux - file and process management, script execution, superuser privileges
  • Programming - control structures, decision points, event callbacks
  • Hardware Interfacing - basic circuits, controlling outputs, receiving inputs

These concepts are wrapped around fun activities. We will learn how to navigate the Linux file system by hunting for files with secret messages. Basic python programming constructs are taught by way of controlling digital circuits (working with buttons, LEDs, buzzers).

Finally, as a capstone to Bootcamp, we build our own fully functioning Simon game complete with code, lights, and buttons.



Early into Bootcamp students will assemble their own Predi-Kit, a hardware bundle that includes a Raspberry Pi and all the accessories. We provide a monitor and keyboard/mouse for use at the academy and that's all you'll need to continue prototyping and exploring at home after Bootcamp concludes. The cost of the kit is included in the Bootcamp tuition and is used throughout subsequent courses taught at Predicate. In fact, many courses will enhance the kit by adding additional sensors or other electronics.

The Predi-Kit includes a Raspberry Pi 2 - Model B, a case for the Pi, USB power adapter, USB wifi adapter, 8GB microSDHC card preloaded with Rasbian and Predicate software, full size breadboard, Pi Cobbler Plus (connects Pi to breadboard), eight 1K resistors, eight LEDs (blue, yellow, green, red), four large tact switches, and male/male jumper cables. The specifics of the kit may change over time due to hardware availability and learning adjustments from prior classes.

Accidents Happen. Blow a LED? Swap inputs and outputs? Corrupt a filesystem? We recognize that accidents happen and are part of the learning process. Predicate will replace any hardware that is damaged in class for the life of the equipment. Break something at home? We'll help identify what is needed to spring back into action.
