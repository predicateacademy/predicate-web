Predicate's Little Conductors

Predicate's Little Conductors is a program specifically designed for our budding innovators (ages 7 and 8) who like to build and are curious about how things work. This class provides a gentle introduction to the foundations of computational thinking - how to think about solving problems with a computer. Little Conductors will simply change the way children play.

We love to code. Little Conductors will learn to basics of coding using a CodeBug wearable computer. Not only is it the cutest electronic product on the market, it is also a very capable learning device. Little Conductors will use a block-based programming language to create basic programs that run directly on the CodeBug.

CodeBug + Snap = The Perfect Match. In addition to the CodeBug, Little Conductors will use Snap Circuits to create fun projects that involve code control of electronics such as lighting an LED, controlling a motor, playing sounds - all from the little CodeBug that could.

Core concepts covered in the 10-week Little Conductors program:

  • Computing Hardware - understanding the CodeBug platform
  • Programming - control structures, decision points
  • Hardware Interfacing - basic circuits, controlling outputs, receiving inputs

Much like our Bootcamp program, these concepts are wrapped around fun activities that introduce a smarter way to play. Little Conductors runs for 10 weeks with 45-minute weekly classes. All hardware is provided for use during the class and the kids get to take the CodeBug home at the end of the program for continued use.
